# Portfolio Website: Sandeep De Silva

This repository contains the source code and assets for my professional portfolio, hosted at https://sandeep7511.github.io/. The site serves as a centralized hub for my academic achievements, data science projects, and research interests in clinical informatics and bioinformatics.

## Overview

The primary objective of this website is to demonstrate a data-driven approach to problem-solving, showcasing a transition from theoretical academic excellence to practical technical implementation. It highlights my expertise in building autonomous systems and managing complex data lifecycles.

## Core Content Pillars

### 1. Data Engineering and ETL
Documentation of end-to-end data pipelines, including the Manchester EPC Analysis project. This section emphasizes my ability to process large-scale datasets (1,000,000+ records) using robust cleaning, indexing, and outlier detection methodologies.

### 2. Autonomous AI Systems
Details on the development of the HARA (Hospital Autonomous Resource Allocation) agent. This showcases the integration of Large Language Models (Gemini API) within decision-making engines for healthcare logistics.

### 3. Clinical and Bio-Informatics
Exploration of predictive modeling in clinical settings—such as mortality prediction in allo-HCT patients—and computational genetics. This reflects a commitment to applying machine learning within the biological and medical domains.

### 4. Academic Milestones
A summary of my progression within the BSc in Applied Data Science Communication program at General Sir John Kotelawala Defence University, including current standing and GPA metrics.
